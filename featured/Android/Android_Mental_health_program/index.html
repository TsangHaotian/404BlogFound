项目介绍

MindBloom 是一个基于 AI 情感计算与积极心理学的心理健康平台，专注于预防性干预和低门槛服务。App 集成了百度文心一言大语言模型，实现智能情绪监测、新闻资讯聚合和日记功能，为用户提供全方位的心理健康支持。这是一个本科小组合作项目。

个人总结

这个项目是我第一次做比较大型的项目，也是从这个项目开始学会了用 GitHub 合作开发。但说实话，从软件本身来看，它对我也就这点帮助了，哈哈。如你所见，这个软件居然是我们 6 个人一起合作开发的，没想到吧？当时第一次做这个项目，觉得自己可牛了，还想着要上架 App、要赚钱，还真给我梦上了。

现在回头看，真是惨不忍睹，写的都是些什么玩意啊。就这还 6 个人一起开发，真是太搞了。不过还是硬着头皮把这个项目重新跑了起来，真的太惨了。这个项目简直就是——什么都有，什么都没有。各种各样的模块堆积在一起，页面写了一大堆，实际上乱得不是半点。登录界面交互逻辑也是拼凑的，整个软件看起来就像一堆毫不相关的拼图被强行凑在了一起。那时候我当组长，还经常为了这个软件开会，现在想起来太好笑了，就这破软件还开会呢。开会那么多次，就做出个这东西？一周还得去老师那边讨论项目进度，全部花在无用功上了。那时候还搞什么跨专业，为了这个软件找了个心理学的同学一起合作，不知道的还真以为自己多专业。最后效果也呈现出来了，就是用了非常大的力气，拉了一坨大的。

现在自己比曾经进步了不少，敢这么羞辱自己，哈哈哈哈哈。说起来那时候还真可爱，人怎么可以一年就变这么多。接着说这个项目的故事吧，这个项目可没少让那个心理学的同学操心，她是一个尽心尽责的好人，但我们那时候实力太弱了，现在都感觉有点对不起她的付出。或者是我那时候能力太弱，拉着我们 6 个人搞了无用功，浪费了大家的时间。再介绍一下吧：新闻模块是我对象不知道哪里找的教程，一步一步写进去的；AI 居然是我们花了 70 多块钱买的项目！但是那时候 DeepSeek 这家公司都还没成立！科技发展得好快好快，就一年的时间，现在更新了那么多高科技。之前这个项目的诞生和现在比起来，简直就像一个乐高做的机器人模型遇到了宇树机器人一样。

而且这个项目最后还吵了不少架。特别是在那年过年的时候，我发现我的组员没有一个人参与项目制作，大家全在混，只有我一个人在用心做。我很生气，直接解散了队伍，然后用 Python 重新做了这个项目，也就是后面的 python_mind_project。后面我重构的那个项目，还通过了软著申请呢。哦对了，那个养成系统也是一坨狗屎，就是几个图片浇水浇肥切换动图，哎哟我去，真是烂到爆炸。新闻界面也是 bug 满天飞，最主要的笔记功能现在也用不了。但是刚刚重新运行项目，发现新闻 API 和 AI 功能居然都是正常的，属实给我有点震惊了，我真没想到啊，一下子就把我拉回到曾经了。

其实骂归骂，那时候的心情还是好怀念啊。哎，不说了，给自己都说感动了。
